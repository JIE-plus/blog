# 排序算法

## 冒泡排序
```java
@Test
    public void test_01() {
        int[] arr = new int[]{23,32,1};
        for (int i = 0; i < arr.length - 1; i++) {
            boolean isSort = false;
            for (int j = 0; j < arr.length - 1 - i; j++) {
                if (arr[j] > arr[j+1]) {
                    int temp = arr[j+1];
                    arr[j+1] = arr[j];
                    arr[j] = temp;
                    isSort = true;
                }
            }
            if (!isSort) {
                break;
            }
        }
        for (int a : arr) {
            System.out.println("a =  " + a);
        }
    }
